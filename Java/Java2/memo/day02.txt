내부 클래스(Inner Class)
	클래스 내부에 클래스를 선언하여 외부 클래스의 필드 접근에
	용이하게 하기 위함.
	내부 클래스의 필드를 사용하기 위해서는 외부 클래스에서
	내부 클래스를 객체화 해야한다.

	외부클래스명 외부객체명 = new 외부생성자();
	외부클래스명.내부클래스명 내부객체명 = 외부객체명.new 내부생성자()

내부 클래스를 사용하는 이유
	A클래스에서 b라는 작업이 자주 쓰이고, 이 작업은 B클래를 만들어야
	쉽게 관리할 수 있다. 하지만 다른 클래스에서 b작업이 필요 없거나
	B클래스를 외부에 노출시키고 싶지 않을 때 사용한다.(캡슐화)
