상속(inheritance)
	1. 기존에 사용중인 클래스의 필드 중
	    새롭게 만들 클래스에서 필요한 것들이 있다면
	    상속을 받고 그대로 사용한다.

	2. 여러 클래스를 선언할 때 중복되는 기능들이
	    존재한다면 공통 기능들을 담아놓을 부모 클래스를
	    선언하고 자식 클래스들에게 상속해준다.




파이썬 2.xx버전
	class Car(object) : 


파이썬 3.xx버전
	class Car() :



class A :
	A필드

class B(A) :
	A, B 필드


	자식 생성자 호출 시 자식 필드만 메모리에 할당된다면
	자식 객체는 부모필드에 절대로 접근할 수가 없다.
	따라서 자식 생성자 호출 시 부모 생성자가 먼저 호출되고,
	만약 부모 필드에서 선언된 메소드의 이름과 자식 필드에서 
	선언된 메소드의 이름이 같다면 자식 생성자 호출 시 부모의
	필드에 자식에서 선언된 기능으로 덮어 씌워진다.
	이것을 재정의(Overriding)이라고 한다.

생성자
	부모 클래스에 생성자가 선언되어 있고,
	자식 클래스에는 기본 생성자가 있다면 부모 클래스 생성자를 자동으로 
	호출해준다. 하지만, 자식 클래스에서 생성자를 직접 선언하면
	부모 생성자를 자식 생성자에서 직접 호출해주어야 한다.



자식클래스에 생성자가 있을 때
	부모에 있는 필드와 자식에서 필요한 필드가 있을 때
	부모 생성자를 호출하여 부모 필드를 초기화 해주고
	추가된 자식 필드를 추가로 초기화 해주어야 한다.




자식클래스에 생성자가 없을 때
	부모의 필드 외에 추가적인 필드가 없을 때 부모 생성자를
	그대로 사용한다.
