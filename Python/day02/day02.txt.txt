제어문자
	
변수
	변수는 저장공간이다.

	x	=	0
     저장공간의	대입	값(상수)
	이름	연산자


자료형	type()
	
정수형	int
실수형	float
문자열	str
	

변수의 선언
	초기화
	변수명 = 초기값

변수 선언 시 RAM
	x = 0
	x라는 이름의 저장공간이  RAM에 할당된다.

주소(C언어)
	저장공간이 RAM에 할당되면 고유한 값인
	주소값이 부여된다.

변수 선언 시 주의사항
	같은 이름으로 선언할 수 있다. 
	(하지만 중복된 이름이 있을 떄에는 선언이 아니라 수정이다.)
	초기화를 반드시 해준다.
	되도록 선언부에 한꺼번에 선언한다.(페이지 상단)


변수명 주의사항
	문자부터 시작한다.
	특수문자는 사용할 수 없다.
	단, _(언더바)는 사용가능하다.
	되도록 소문자를 사용한다.	
	공백은 사용할 수 없다.

	good_boy
	goodBoy : 카멜표기법, 헝가리안 표기법


	되도록 뜻이 있는 단어를 선택한다.
	a, b, c, d, e,...(x)
	data, number, seq, name, age, cnt,...(O)
	
	되도록 한글은 사용하지 않는다.

	
변수의 사용
	변수명을 사용한다.
	
	x = 10		저장공간
	print(x)		값
	x+8		값
	x = 20.9+x	저장공간
