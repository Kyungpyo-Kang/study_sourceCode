파일 입출력
	파일객체 = open("경로", "목적")

	목적
	-w : write 경로에 파일이 없으면 생성(수정 삭제)
	-a  : add 경로에 파일이 없으면 생성(추가)
	-r  :  read 경로에 파일이 없으면 오류(목록)


	w : 마지막에 쓴 내용으로 덮어 씌워진다.
	a  : 맨 아래부터 내용이 추가된다.

	입력하기
		파일객체명.write("문자열")

	출력하기
		파일객체명.readlines()


	close() : 버퍼를 비워주어야 사용에 적용된다.
	반드시 작업이 끝나면 파일객체명.close()를 사용!
