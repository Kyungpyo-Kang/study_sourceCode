메소드 선언시 주의사항
	메소드 내에서 메소드를 선언하지 않고
	메소드를 사용할 때에는 보통 메소드 안에서 사용한다.
	매개변수가 없을 때에는 사용시 값을 전달할 수 없다.
	리턴값이 있다면 사용하는 부분 통채로를 리턴값으로 본다.

메소드 선언 순서
예) 두 정수의 덧셈 메소드 만들기

1. 메소드명을 생각한다.
	add, plus, 더하기, onePlusOne

	def add() :

2. 매개변수를 생각한다.

	def add(num1, num2) :

3. 실행할 문장을 생각한다.


	def add(num1, num2) :
		print("두 정수의 덧셈 기능입니다.")
		
4.  리턴값을 생각한다.

	def add(num1, num2) :
		print("두 정수의 덧셈 기능입니다.")
		return num1+num2


리턴을 해야할 때
	메소드 내에서 모든 작업이 끝날 수 없을 때
	사용하는 부분에서 메소드 내에서의 결과를 알려주어야 할 때


리턴을 안해도 될 때
	메소드 내에서 모든 작업이 완료 될 때
	사용하는 부분에 결과를 알릴 필요가 없을 때

	    메소드로 선언한다. 절대 특정성을 부여해서는 
	    안된다.
